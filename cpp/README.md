# C++ Beginner Programs

This folder contains a few simple C++ programs you can use to practice.

## Building and Running

You can build any of the programs using `g++` from the command line. For example:

```bash
g++ hello_world.cpp -o hello_world
./hello_world
```

Replace `hello_world.cpp` with the file you want to compile.

### Files

- `hello_world.cpp` - Prints "Hello, world!"
- `sum_numbers.cpp` - Reads two numbers from the user and prints their sum.
- `guess_number.cpp` - A small number guessing game.
- `factorial.cpp` - Computes the factorial of a given integer.
